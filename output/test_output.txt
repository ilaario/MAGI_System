MAGI SYSTEM TEST SUITE
============================================================

This test runs multiple decision scenarios through the MAGI system.
Each scenario is evaluated by three specialized agents:

Agents:
- Melchior   → analytical / risk-balanced reasoning
- Balthasar  → human / ethical impact reasoning
- Casper     → strategic / pragmatic reasoning

Models used:
- Melchior   → openai/gpt-4o
- Balthasar  → anthropic/claude-3-sonnet
- Casper     → google/gemini-2.5-flash-lite
- Auditor    → openai/gpt-4o-mini

The system will:
- Collect individual agent decisions
- Compute majority and weighted outcomes
- Run consistency checks and audits when needed
